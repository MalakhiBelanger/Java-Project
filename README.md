# E-Sports Team Management Software
_- By Malakhi Lemieux-Belanger and Elijah Harnadek_

## What the software manages  
This software helps esports team managers view 
and edit data associated with their teams players.
Each manager has login info that ties them to their
respective team that they manage.  

## Login and Registration Page  
Allows the user to register and login with their own credentials and saves all of their data to the cloud  

## Homepage  
Welcomes the user with their username into the program  

## Games  
Allow the user to add and view games with a customizable image hosted via an image web url  

## Players  
Allow the user to manager their players, adding and deleting the players via input  

## Finances  
User can add finances that they might forget so they are reminded to pay them  
(Scrapped idea was to pay players through the finances page and the sum will be shown in the players net income section)  

## Stats  
Simple widget display showing the users teams overall data

## Different utility  
This software doubles as a player overview if the user
signing in is a player. The software displays the respective
users stats and data in the games that they play, and can sign up
or change games they participate in (Notifies manager first
for confirmation).

## Matches Display  
Show individual match instances that players participated in, shows the stats of the game including score, game type, and players.
